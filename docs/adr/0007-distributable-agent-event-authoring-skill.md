# Distributable Agent Event Authoring Skill

The repository will maintain `skills/rpgmaker-event-authoring` as a distributable agent skill for tool users. The skill exists to help agents take over RPG Maker MV event authoring from initial Workspace setup through Event DSL source creation, validation, Diff review, and optional Push, with authoring judgment as the center and Workspace workflow as support. This keeps the agent-facing authoring behavior versioned with the project instead of scattering it across external prompts or treating it as ordinary prose documentation.
